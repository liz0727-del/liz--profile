# 项目文件夹结构说明

本文档解释了项目中每个文件夹的用途，方便你日后存放视觉资产。

---

## 📁 根目录结构

```
个人网站搭建/
├── public/              # 静态公共资源
├── src/                 # 核心源代码
│   ├── assets/          # 视觉资产 (图片/视频/字体)
│   │   ├── images/      # 图片文件
│   │   ├── videos/      # 视频文件
│   │   └── fonts/       # 自定义字体
│   ├── components/      # React 组件
│   └── hooks/           # 自定义 React Hooks
├── plan.md              # 开发计划
└── .cursorrules         # AI 规则
```

---

## 📂 文件夹详解

### `public/`
**用途**: 存放不需要被 Vite 处理的静态文件。
- 网站的 `favicon.ico` (浏览器标签图标)。
- 任何需要直接通过 URL 访问的文件 (如 `robots.txt`)。
- **你的操作**: 一般不需要手动放东西，我们会自动生成。

---

### `src/assets/images/`
**用途**: 存放你的作品集图片。
- 项目截图、个人头像、背景图等。
- **支持格式**: `.jpg`, `.png`, `.webp`, `.svg`。
- **你的操作**: 把你未来的作品图片拖到这里即可。

---

### `src/assets/videos/`
**用途**: 存放视频素材。
- 项目演示视频、动态背景等。
- **支持格式**: `.mp4`, `.webm`。
- **你的操作**: 把需要展示的视频拖到这里。

---

### `src/assets/fonts/`
**用途**: 存放自定义字体文件。
- 如果你有特殊的本地字体 (如购买的商业字体)，放在这里。
- **支持格式**: `.woff2`, `.woff`, `.ttf`, `.otf`。
- **你的操作**: 大部分字体我们会用 Google Fonts CDN，但如果需要离线字体，放这里。

---

### `src/components/`
**用途**: 存放所有 React 组件。
- 每个 UI 模块 (Header, Card, AIChatWidget) 都是一个组件。
- **你的操作**: 你不需要手动创建，代码由我们一起写。

---

### `src/hooks/`
**用途**: 存放自定义 React Hooks (逻辑复用)。
- 例如 `useMousePosition` (获取鼠标位置)、`useScrollProgress` (获取滚动进度)。
- **你的操作**: 你不需要手动创建，代码由我们一起写。

---

## 下一步
文件夹已就位。当你准备好视觉素材时，直接拖入对应文件夹即可。
